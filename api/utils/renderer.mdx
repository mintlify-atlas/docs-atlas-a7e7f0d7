# Renderer

The renderer provides a high-level DirectX 9 rendering interface for drawing shapes, text, gradients, and other visual elements in CS:GO.

## Initialization

### init

```cpp
void g_render.init(IDirect3DDevice9* buffer_device);
```

Initializes the renderer with a DirectX device.

<ParamField path="buffer_device" type="IDirect3DDevice9*">
  DirectX 9 device to use for rendering
</ParamField>

<Note>
  Call this once during SDK initialization, typically in the EndScene hook.
</Note>

---

## State Management

### setup_state / finish_state

```cpp
void g_render.setup_state();
void g_render.finish_state();
```

Manages DirectX render state for custom drawing.

<CodeGroup>
```cpp Render Loop Pattern
void on_frame() {
    g_render.setup_state();
    
    // Your rendering code here
    g_render.render_text(10, 10, AL_DEFAULT, FLAG_OUTLINE, "Hello", font, color::white());
    
    g_render.finish_state();
}
```
</CodeGroup>

## Font Management

### create_font

```cpp
void g_fonts.create_font(
    std::uint32_t hash,
    std::size_t size,
    std::size_t weight,
    bool anti_aliased,
    const char* font_name
);
```

Creates a font for text rendering.

<ParamField path="hash" type="std::uint32_t">
  Unique hash identifier for this font (use FNV1A hash)
</ParamField>

<ParamField path="size" type="std::size_t">
  Font size in pixels
</ParamField>

<ParamField path="weight" type="std::size_t">
  Font weight (e.g., FW_NORMAL, FW_BOLD)
</ParamField>

<ParamField path="anti_aliased" type="bool">
  Enable anti-aliasing for smoother text
</ParamField>

<ParamField path="font_name" type="const char*">
  System font name (e.g., "Verdana", "Arial")
</ParamField>

Retrieves a font by its hash identifier.

## Enumerations

### Font Flags

```cpp
enum font_flags : unsigned {
    FLAG_NONE,
    FLAG_DROPSHADOW,
    FLAG_OUTLINE,
    FLAG_OUTLINE_SEMI
};
```

<ResponseField name="FLAG_NONE" type="0">
  No special rendering
</ResponseField>

<ResponseField name="FLAG_DROPSHADOW" type="1">
  Draw text with a drop shadow
</ResponseField>

<ResponseField name="FLAG_OUTLINE" type="2">
  Draw text with a full outline
</ResponseField>

<ResponseField name="FLAG_OUTLINE_SEMI" type="3">
  Draw text with a semi-transparent outline
</ResponseField>

---

### Font Alignment

```cpp
enum font_alignment : unsigned {
    AL_DEFAULT           = (1 << 0),
    AL_VERTICAL_TOP      = (1 << 1),
    AL_VERTICAL_CENTER   = (1 << 2),
    AL_HORIZONTAL_LEFT   = (1 << 3),
    AL_HORIZONTAL_CENTER = (1 << 4),
};
```

Alignment flags can be combined with bitwise OR.

## Example Usage

<CodeGroup>
```cpp Basic Drawing
// Draw a filled box with outline
g_render.render_filled_rectangle(100, 100, 200, 50, color(30, 30, 30, 200));
g_render.render_rectangle(100, 100, 200, 50, color::white());
```

```cpp Text Rendering
// Create and use a font
g_fonts.create_font(FNV1A("main_font"), 14, FW_NORMAL, true, "Verdana");
auto font = g_fonts[FNV1A("main_font")];

g_render.render_text(
    10, 10,
    AL_DEFAULT,
    FLAG_OUTLINE,
    "HotWheels SDK",
    font,
    color::white()
);
```

```cpp Gradients
// Horizontal gradient
g_render.render_gradient<gradient_type_t::HORIZONTAL>(
    50, 50, 300, 100,
    color(255, 0, 0),    // Red
    color(0, 0, 255)     // Blue
);
```

```cpp Health Bar
void draw_health_bar(int x, int y, int width, int height, float health_pct) {
    // Background
    g_render.render_filled_rectangle(x, y, width, height, color(0, 0, 0, 150));
    
    // Health fill
    int health_width = static_cast<int>(width * health_pct);
    g_render.render_filled_rectangle(x, y, health_width, height, color(0, 255, 0));
    
    // Border
    g_render.render_rectangle(x, y, width, height, color::white());
}
```
</CodeGroup>
