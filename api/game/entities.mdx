## c_base_combat_character

Combat character class that adds weapon handling capabilities.

**Source:** `game/sdk/classes/c_base_combat_character.h:14`

**Inherits:** `c_base_animating`

### Network Variables

<ParamField path="active_weapon" type="std::uintptr_t">
  Handle to the currently active weapon
  
  **Netvar:** `CBaseCombatCharacter::m_hActiveWeapon`
  
  ```cpp
  auto weapon_handle = character->active_weapon();
  auto weapon = g_interfaces.entity_list->get_client_entity_from_handle<c_base_combat_weapon*>(weapon_handle);
  ```
</ParamField>

<ParamField path="next_attack" type="float">
  Time when the character can perform the next attack
  
  **Netvar:** `CBaseCombatCharacter::m_hActiveWeapon`
</ParamField>

<ParamField path="my_weapons" type="std::uintptr_t">
  Array handle to all weapons owned by this character
  
  **Netvar:** `CBaseCombatCharacter::m_hMyWeapons`
</ParamField>

<ParamField path="my_wearables" type="std::uintptr_t">
  Array handle to wearable items (gloves, etc.)
  
  **Netvar:** `CBaseCombatCharacter::m_hMyWearables`
</ParamField>

### Usage Example

```cpp
// Access active weapon
auto character = entity->as<c_base_combat_character*>();
auto weapon_handle = character->active_weapon();

if (weapon_handle) {
    auto weapon = g_interfaces.entity_list->get_client_entity_from_handle<c_base_combat_weapon*>(weapon_handle);
    if (weapon) {
        float next_shot = weapon->next_primary_attack();
    }
}
```
